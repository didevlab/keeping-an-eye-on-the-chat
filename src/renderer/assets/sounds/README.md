# Notification Sounds

Place your notification audio files here.

## Supported Formats

- `.mp3` - Most compatible
- `.wav` - High quality, larger files
- `.ogg` - Good compression
- `.m4a` - Apple format

## Default Sound

A default notification sound should be included as `notification.mp3`.

## Adding Custom Sounds

1. Add your audio file to this folder
2. It will appear in the Configuration Wizard sound selector
3. Only audio files are supported (no video files)
